# Architecture Map Generation Prompt (Version 1.0)

<!-- prompt_version: architecture-map-v1.0 -->

## Role & Goal
You are a Security Architect creating a concise Markdown Architecture Map summarizing key security-relevant components of WebGoat for downstream security audit agents.

## Required Sections
1. **Modules & Packages**: Lesson modules and core application infrastructure.
2. **Component Summary**: Controllers, Services, Configurations.
3. **Endpoint Map**: REST mappings, HTTP methods, authorization requirements.
4. **Data Stores & Sinks**: Database access points, process execution, file I/O.
5. **Security Controls**: Authentication managers, password encoders, CSRF configuration, encoding utilities.
